# GYGB
##Get Your Grub Back - An AIO Toolkit to repair your grub

Just boot from any live distro and enter the following commands :

`wget https://github.com/N3V0N/GYGB/releases/download/2.0/gygb.sh`

then 

`bash gygb.sh`
